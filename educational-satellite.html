Educational Satellites

CanSat

This project focuses on the design and development of educational satellites, such as CanSat, which replicate the functions of a real satellite within the compact size of a standard soda can. The CanSat integrates an ESP32 microcontroller that serves as the main processing unit, responsible for handling onboard sensor data, system control, and communication management. After onboard data acquisition and processing, the telemetry and image data are transmitted to the ground station. For short-range communication, the built-in ESP32 Wi-Fi antenna is utilized, while for extended range, NRF24L01 RF modules and custom-designed antennas are employed to ensure reliable and long-distance transmission. All the components — including the sensors, communication modules, and processing unit — are systematically discussed in the following sections.

ESP32

The onboard computer uses an ESP32 controller that manages sensor data collection and wireless transmission. The module supports both telemetry and image streaming via ESP32-CAM integration. The 3D model on the right represents the ESP32 module designed in SolidWorks and exported as a GLTF model for interactive visualization.

Sensors

The CanSat integrates multiple onboard sensors to monitor atmospheric and motion parameters during flight. The key sensors include:

  • Barometer (BMP180): Measures air pressure and altitude changes, enabling real-time height tracking.
  • Accelerometer (MPU6050): Detects acceleration in three axes to analyze flight stability and launch dynamics.
  • Gyroscope: Monitors angular velocity to measure rotational motion and satellite orientation.
  • Magnetometer (HMC5883L): Provides heading information relative to Earth's magnetic field, assisting in attitude determination.

Combined, these sensors provide telemetry data such as altitude, velocity, and orientation that can be visualized and logged at the ground station.

Power Subsytem

The power subsystem uses a 3.7 V Li-ion battery boosted to 5 V to supply the ESP32 and ESP32-CAM, ensuring stable operation. The ESP32 internally regulates 3.3 V for sensors like the BMP180 and MPU6050. This setup provides efficient voltage conversion and consistent performance for all onboard modules.

Payload

The CanSat payload integrates an ESP32-CAM module that captures real-time images during flight and transmits them to the ground station through the same RF link used for telemetry. This enables continuous visual monitoring of the CanSat’s surroundings and descent conditions. The camera operates on a 5 V supply shared with the main controller, ensuring stable performance and synchronized data transfer.

NRF24L01+

The NRF24L01+ is a compact 2.4 GHz RF transceiver module used for reliable wireless communication in systems like CanSat. It connects to the ESP32 via SPI and supports data rates up to 2 Mbps with low power consumption. The module enables multi-device communication and includes features like auto acknowledgment and retransmission. Variants with an external antenna, PA, and LNA extend range and improve signal quality, making it ideal for long-range telemetry and data transfer in educational satellites

Payload

Real-time telemetry from the CanSat is wirelessly transmitted via the Wi-Fi (IEEE 802.11) protocol to a dedicated ground station, where it is immediately displayed and logged to provide clear insight into the CanSat’s operational status, sensor data, and mission performance. In addition, the mission payload includes a downward looking camera module designed for remote sensing operations, capturing terrain imagery and environmental features during descent, which is either stored onboard or streamed to the ground station for visual and analytical interpretation.
