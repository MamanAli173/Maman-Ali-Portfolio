RF & Microwave Components

Waveguide Filter

This project focuses on a waveguide-based bandpass filter for X-Band applications using the standard WR-90 waveguide (dimensions: 22.86 mm × 10.16 mm). The designed filter achieves a bandwidth of 200 MHz, optimized for high selectivity and minimal insertion loss across the operating band.

Adapter

The first component is a coax-to-waveguide adapter, which allows RF signals to transition from coaxial cables into the rectangular waveguide with minimal reflection and loss. This component ensures proper impedance matching and smooth signal propagation into the waveguide.

Filter Bottom Cover

The bottom cover of the filter contains irises with optimized lengths and widths to achieve the desired bandpass characteristics. These dimensions are carefully tuned to control coupling and resonance, providing efficient filtering within the 200 MHz bandwidth.

Future Work

As a future extension of this project, multiple waveguide bandpass filters can be combined to design a multiplexer. This multiplexer would allow simultaneous transmission or reception of different frequency channels within the X-Band, increasing system capability and enabling advanced RF communication applications.
